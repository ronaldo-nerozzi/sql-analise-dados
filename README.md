# 📊 SQL — Análise de Dados Públicos Brasileiros

Repositório com consultas SQL desenvolvidas para análise de dados públicos do Brasil, com foco em extração, transformação e geração de insights relevantes.

---

## 🎯 Objetivo

Demonstrar habilidades práticas em SQL, incluindo:
- Consultas simples e complexas (SELECT, WHERE, GROUP BY, ORDER BY)
- Junções entre tabelas (JOIN)
- Funções de agregação (COUNT, SUM, AVG, MAX, MIN)
- Subconsultas e CTEs
- Análise exploratória de dados via SQL
